An `nvmrc` file is included if using [nvm](https://github.com/creationix/nvm).
- Node 6.13.0
- Apache Cassandra 

## Development

### Installing Dependencies

From within the root directory:
```sh
npm install -g webpack
npm install
```
### Scripts
Start the server
```
npm start
```
Build the bundle.js
```
npm run build
```
Seed the database with dummy data
```
npm run seed
```
